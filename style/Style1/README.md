# Style 1 - Style Original

Ce dossier contient le style CSS original de l'application exam-platform.

## Fichiers
- `App.css` - Styles de l'application principale
- `index.css` - Styles globaux et composants de base

## Couleurs principales
- Primary: #4f46e5 (Indigo)
- Secondary: #10b981 (Emerald)
- Danger: #ef4444 (Red)
- Warning: #f59e0b (Amber)

## Pour restaurer ce style
Copiez les fichiers CSS dans le dossier `src/` :
```bash
copy style\Style1\App.css src\App.css
copy style\Style1\index.css src\index.css
```
